Projects

Spellshock AR

  • Watch Video

This game is your typical rhythm game with a twist: Tap the orbs with colors that are different than the bottom sensor and allow the orbs with the same color to pass. All while doing it in AR. I developed this app using Unity and Rhythm tool to analyze music files for beats and onsets.

VR Escape Room

  • Watch Video

This program was created to educate about permutations and combinations in the form of an Escape Room type game. The game starts off within a jail cell. The player is given a certain amount of time to figure out the three digit permutation. The next stage consists of colors where the player will have to guess the color combination to complete the game.

Wildfire VR

  • Watch Video

This program is created to help simulate a wildfire evacuation drill and demonstrate what would happen if a wildfire were to occur. The program starts off with the user inside a house where a phone will ring telling you to check the news After, the user will be instructed by an officer to follow the evacuation route and head to the safety point.
